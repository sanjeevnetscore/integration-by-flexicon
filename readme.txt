== Description ==
Integration by Flexicon is a lightweight contact form plugin that redirects the user to a custom plugin page after installation. It allows users to submit their Name, Company Email, Mobile Number, and Message through a simple form. Upon submission, an email is sent to the WordPress site admin as well as to the submitter. Designed with simplicity and clarity in mind, it ensures seamless communication between businesses and the Flexicon team.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/integration-by-flexicon` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Upon activation, you will be redirected to the plugin’s custom page containing the contact form.

== Frequently Asked Questions ==

= Where does the form send the submitted data? =
The form sends the details to the site admin’s email and also replies to the submitter’s email.

= Can I change the recipient email address? =
Currently, the form sends to the site admin email. You can customize the plugin code to change it if needed.
